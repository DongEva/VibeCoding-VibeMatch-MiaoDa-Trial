# ✨ Magic Closet-MiaoDa (秒搭——世界如此百搭) - 智能时尚搭配助手

这是一个基于 Web 的移动端优先的智能时尚搭配应用。利用 AI 技术，将用户上传的单品转化为极具时尚杂志感的全身穿搭效果图。

本项目注重视觉美学，采用复古杂志风格（Vintage Magazine Style），并针对移动端设备（1206x2622分辨率）进行了深度优化。

Miaoda is a visual styling tool focused on "hard-to-match clothing items" — its core value isn’t recommending new clothes, but unlocking the full matching potential of one piece you already own.

Simply upload a photo of a garment (no need to fill in style, body type, or occasion details), and our AI stylist will generate 3-4 complete outfits. The results page features magazine-style full-body images: 1 primary outfit (safe, wearable, and eye-catching, taking up 60-70% of the screen) plus 2-3 inspiration looks (for different moods or occasions, displayed as clickable cards).

A small thumbnail of your uploaded item is included for easy reference. No complicated features, just straightforward, visually appealing solutions to answer "How else can I wear this?" and help you make the most of every piece in your wardrobe.

## 🚀 快速开始 (Quick Start)

1.  **配置密钥**：
    打开 `config.js` 文件，找到 `API_CONFIG` 部分，将 `YOUR_API_KEY_HERE` 替换为你申请的有效 API Key。

    ```javascript
    const API_CONFIG = {
        API_KEY: 'sk-xxxxxxxxxxxxxxxxxxxxxxxx', // 替换此处
        // ...
    };
    ```

2.  **运行项目**：
    由于涉及到本地文件读取和摄像头权限，建议使用本地服务器运行：
    *   **Python**: `python -m http.server 8000`
    *   **VS Code**: 使用 "Live Server" 插件打开 `simple-fashion-generator.html`。

3.  **手机访问**：
    确保手机与电脑连接同一 Wi-Fi，通过电脑 IP 地址访问（例如：`http://192.168.1.x:8000/simple-fashion-generator.html`）。

## 📅 项目演进历程 (Evolution History)

本项目从一个简单的功能原型，逐步进化为体验完善的“掌上时尚杂志”，主要经历了以下几个阶段：

### Phase 1: 视觉奠基 (Aesthetic Foundation)
*   **风格确立**：定调为“高端复古时尚杂志”。使用了复古金 (`#B8860B`) 和焦糖色 (`#8B4513`) 作为主色调，配合纸质纹理背景。
*   **排版优化**：引入思源黑体与衬线字体，强化大标题 (`h1`) 与正文的对比，营造杂志封面的视觉冲击力。

### Phase 2: 移动端适配 (Mobile Optimization)
*   **响应式布局**：针对 `1206x2622` 的竖屏比例进行优化，使用 `vh/vw` 和 `clamp()` 确保在不同手机屏幕上布局不崩坏。
*   **交互细化**：
    *   将“上传”功能拆分为**“拍照”**与**“相册”**两个独立按钮，符合移动端操作直觉。
    *   按钮位置使用 CSS 变量精确控制，确保手指易于点击。

### Phase 3: 功能增强 (Feature Expansion)
*   **灵感档案 (Archive)**：将原本简单的历史记录升级为“灵感档案”，以卡片形式展示过往搭配，支持日期回溯。
*   **动态背景**：首页引入循环播放的时尚视频 (`home-bg.mp4`)，配合“一键开启”按钮，提升应用的“待机”质感。
*   **智能标签**：预设“多巴胺”、“新中式”、“Old Money”等热门风格标签，简化用户输入。

### Phase 4: 性能与体验打磨 (Polishing)
*   **动画优化**：修复了 CSS 动画在移动端浏览器（如 iOS Safari）上的兼容性问题（Magnifier 动画）。
*   **无缝加载**：重构了图片加载逻辑，引入 `Promise` 预加载机制，消除了结果页切换时的“旧图闪烁”问题，确保视觉体验丝滑流畅。
